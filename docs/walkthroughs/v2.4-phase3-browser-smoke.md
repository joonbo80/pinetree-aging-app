# v2.4 Phase 3 Browser Smoke

Date: 2026-05-15
Status: READY FOR J BROWSER VERIFICATION
Scope: C5-C11 SharePoint-backed persistence and workflow features

## 1. Purpose

This smoke test verifies the v2.4 Phase 3 workflow chain end to end:

```text
Entra sign-in
-> SharePoint snapshot upload
-> SharePoint snapshot read-back
-> Workflow list metadata
-> Owner/Memo
-> Row indicators
-> Promise tracking
-> Email draft
```

The test intentionally uses J's browser session because Microsoft Entra
and SharePoint permissions depend on the real Pinetree tenant and user
context.
